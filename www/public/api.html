/network/ecpm

Generate a report summarizing the network eCPM and comparing to your eCPM across the specified breakdowns.

Authorizations:
NoneAPI Key
query Parameters

required
any
Example: breakdowns=/api/network/ecpm?breakdowns=week,format

This is a comma separated list of breakdowns. Valid values include: "week", "genre", "country", "format", "platform", "mobile_os_version", "serving_restriction", "gma_sdk_version".

/account

The account status can be one of: STARTED_BACKFILL, FINISHED_BACKFILL, or ERROR. When the user first creates an account, they start in the STARTED_BACKFILL state until we finish importing their data. Then, it transitions into FINISHED_BACKFILL. If at any point, we are unable to load their data from AdMob, it transitions into ERROR and stays their until the issue is resolved.

Authorizations:
NoneAPI Key

Responses
